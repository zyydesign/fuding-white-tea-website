# 福鼎白茶数字展示网站

“基于Web前端技术的福鼎白茶数字展示网站设计与实现”项目的网站文件。

## 页面结构
- index.html 首页
- heritage.html 非遗脉络
- craft.html 工艺与茶品
- travel.html 茶旅线路
- experience.html 互动体验

## 更新内容
1. 整合茶山、采茶、茶品与视频素材，完成多媒体展示。
2. 以真实图片和视频模块呈现首页与各内页内容。
3. 工艺页包含流程展示与茶品图谱模块。
4. 茶旅页新增路线视觉卡片和视频场景模块。
5. 互动页海报生成器改为真实背景图切换。

## 运行方式
双击 index.html

## 文件说明
- assets/style.css 页面样式
- assets/main.js 页面交互
- assets/media/ 图片与视频素材
